# Workplace Obsidian Vault

This repository contains a starter Obsidian vault designed for workplace productivity. It includes templates, dashboards, and workflows to help teams capture, organize, and act on knowledge efficiently.

## ðŸš€ Getting Started

1. Clone this repository and remove git tracking:

```bash
git clone https://github.com/AYaroslavskiy91/obsidian-starter-vault.git && cd obsidian-starter-vault && rm -rf .git
```
2. Open the folder in Obsidian as a vault.

3. (Optional) Go to [Api Keys](https://aistudio.google.com/app/apikey) to get your API key to integrate google gemini (or any agent) into your vault. Copy the API key for it. Press `cmd+,` to bring up your settings. Navigate to Text Generator, and paste the API key into the appropriate textbox.

4. Start by creating a Daily Note via `Cmd+p > return`.
